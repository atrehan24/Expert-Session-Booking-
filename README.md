
# Expert Session Booking Web Application

A full-stack Booking Session Web Application that allows users to book expert sessions seamlessly. Built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

## Features

- User registration and login
- Book expert sessions
- View and manage bookings
- Backend API integration
- MongoDB database storage
- Responsive frontend UI

## Tech Stack

Frontend:
- React.js
- JavaScript
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB (MongoDB Atlas)

Deployment:
- GitHub

## Installation & Setup

### Clone the repository

```
git clone https://github.com/atrehan24/Expert-Session-Booking-.git
```

###  Install dependencies

Frontend:

```
cd client
npm install
```

Backend:

```
cd server
npm install
```

###  Add Environment Variables

Create a `.env` file in the server folder and add:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

### Run the project

Backend:

```
npm start
```

Frontend:

```
npm start
```
